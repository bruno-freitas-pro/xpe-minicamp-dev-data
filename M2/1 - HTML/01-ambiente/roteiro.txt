Roteiro - Ambiente
==================

(X) Falar rapidamente sobre o ambiente Windows x Linux x MacOS
(X) Instalar a fonte JetBrains Mono - https://www.jetbrains.com/lp/mono/
(X) Baixar VSCode no formato compactado e portável (.zip) - https://code.visualstudio.com/
(X) Para verificar se o Windows é 32 ou 64 bits - https://superuser.com/a/321995
(X) Instalar e configurar VSCode Portable - https://code.visualstudio.com/docs/editor/portable
(X) Instalar as extensões "Prettier", "vscode-icons", "Winter is Coming Theme" e, principalmente, "Live Server"
(X) Inserir conteúdo do arquivo "settings.json" no VSCode
(X) Criar pasta "projetos\01-ambiente" no Desktop
(X) Criar um app HTML simples (Hello, world!) na pasta "01-ambiente" e demonstrar como o trabalho é "pouco produtivo" com os diversos "refreshes" da página
(X) Demonstrar melhoria de produtividade com a utilização da extensão "Live Server"
